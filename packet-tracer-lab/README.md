## 7. Packet Tracer Network Security Labs
**Tools:** Cisco Packet Tracer, IOS CLI

Six hands-on network simulations covering enterprise networking and security concepts built in Cisco Packet Tracer using IOS CLI configuration.

| Lab | Key Concepts |
|-----|-------------|
| [VLAN Segmentation & ACLs](packet-tracer-labs/VLAN_Segmentation_ACL_Lab/writeup.md) | VLANs, trunking, inter-VLAN routing, DHCP, DNS, ACLs |
| [DMZ Architecture](packet-tracer-labs/DMZ_Architecture_ACL_Lab/writeup.md) | Zone separation, web server exposure, ACL enforcement |
| [Layer 2 Security](packet-tracer-labs/Layer2_Security_PortSec_DHCP_DAI/writeup.md) | Port security, DHCP snooping, Dynamic ARP Inspection |
| [Site-to-Site IPSec VPN](packet-tracer-labs/Site_to_Site_VPN_IPSec_Lab/writeup.md) | IPSec, ISAKMP, crypto maps, tunnel configuration |
| [OSPF Dynamic Routing](packet-tracer-labs/OSPF_Dynamic_Routing_Lab/writeup.md) | OSPF neighbor adjacency, dynamic route advertisement |
| [NAT/PAT Simulation](packet-tracer-labs/NAT_PAT_Internet_Simulation_Lab/writeup.md) | Address translation, inside/outside NAT, PAT overload |

**Key Skills Demonstrated:**
- IOS CLI configuration across routers and switches
- Network segmentation and access control enforcement
- Layer 2 and Layer 3 security implementation
- VPN and encryption concepts
- Dynamic routing protocol configuration
- Network address translation

**What I Learned:**
How enterprise networks are built, secured, and managed at multiple layers — from physical port security up through routing, VPNs, and NAT. These labs directly map to Security+ exam domains and real-world help desk and SOC scenarios.
