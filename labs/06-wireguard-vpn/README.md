# WireGuard VPN Server

**Tools:** Raspberry Pi 5, WireGuard, Linux CLI, UFW Firewall  
**Year:** 2025

## What I Built
Self-hosted a WireGuard VPN server on a Raspberry Pi 5 to encrypt traffic and practice secure remote access — the same concept used in enterprise environments for remote workforce connectivity.

## What I Did
Generated public/private key pairs for the server and client, configured peer settings, and managed firewall rules with UFW to control VPN traffic. Set up the client config and validated the encrypted tunnel was working. Handled all configuration through the Linux CLI.

## What I Learned
Working through the key exchange and peer config manually gave me a solid understanding of how VPNs actually work under the hood — not just clicking connect on a client app. It also reinforced Linux firewall management in a way that felt practical rather than academic.
